🏠 Service Hub – Full Stack Web Application
Python (Flask) + MySQL Backend
-------------------------------------------
📌 Overview

Service Hub is a full-stack web application designed to connect customers with skilled service providers such as electricians, plumbers, and mechanics  etc.

The platform allows users to register, log in, search for workers, and view profiles — all powered by a Flask backend and MySQL database.

It demonstrates core concepts of web development including authentication, database design, session management, and REST APIs.
-------------------------------------------
⚙️ Features

1. User Registration & Login system

2. Secure password storage using hashing

3. Browse workers by profession

4. Search workers by name or skill

5. Worker registration system

6. User profile page (protected route)

7. Flash messages for user feedback

8. Responsive UI design.
-------------------------------------------
📌 Summary

Service Hub demonstrates how:

Backend (Flask) handles logic

Database (MySQL) stores structured data

Frontend (HTML/CSS) displays content

APIs enable data exchange

Together, they form a complete full-stack web application.

Finally:-

The Service Hub is a full-stack web application built using Flask and MySQL that connects users with skilled service workers like electricians, plumbers, carpenters, and mechanics. It includes user authentication, worker registration, service browsing, and a responsive UI, enabling users to easily find and contact professionals.
